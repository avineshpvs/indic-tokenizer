=============================================
indic-tokenizer: Tokenizer for Indian Scripts
=============================================

How to use ??
=============

.. parsed-literal::

    python indic_tokenizer --i inputfile --o outputfile --l hin

    --i input     <input-file>
    --o output    <output-file>
    --l language  select language (3 letter ISO-639 code)
			Hindi       : hin
			Telugu      : tel
			Tamil       : tam
			Malayalam   : mal
			Kannada     : kan
			Bengali     : ben
			Oriya       : ori
			Punjabi     : pan
			Marathi     : mar
			Nepali      : nep
			Gujarati    : guj
			Bodo        : bod
			Konkani     : kok
			Assamese    : asm
